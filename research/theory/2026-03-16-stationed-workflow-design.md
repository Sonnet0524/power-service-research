## 📋 文档概览

**目标**：设计从计划到总结的完整驻点工作流程

**适用范围**：武侯供电中心现场驻点工作

**流程周期**：单次驻点周期（驻点前1天 → 驻点当天 → 驻点后1周）

---

## 一、流程总览

### 1.1 全流程架构

```
┌─────────────────────────────────────────────────────────────┐
│                     驻点工作全流程                           │
└─────────────────────────────────────────────────────────────┘
                              ↓
        ┌─────────────────────┼─────────────────────┐
        ↓                     ↓                     ↓
┌───────────────┐    ┌───────────────┐    ┌───────────────┐
│  驻点前阶段   │    │  驻点中阶段   │    │  驻点后阶段   │
│               │    │               │    │               │
│  T-1天        │    │  T日          │    │  T+1~T+7天    │
└───────────────┘    └───────────────┘    └───────────────┘
        ↓                     ↓                     ↓
   信息准备            现场执行              跟踪总结
```

### 1.2 流程阶段划分

| 阶段 | 时间 | 核心任务 | 关键产出 |
|------|------|---------|---------|
| **驻点前** | T-1天 | 信息准备、任务规划 | 驻点工作计划、重点客户清单 |
| **驻点中** | T日 | 现场执行、数据采集 | 工作记录、现场照片、客户反馈 |
| **驻点后** | T+1~T+7天 | 跟踪回访、问题整改、总结归档 | 跟踪记录、整改报告、经验总结 |

### 2.2 详细步骤

#### 步骤1：查询驻点小区信息

**执行时间**：驻点前1天上午

**操作内容**：
1. 打开WPS多维表"小区台账"
2. 查询目标小区的基础信息
3. 记录关键信息

**查询内容清单**：

| 信息类型 | 具体内容 | 用途 |
|---------|---------|------|
| **基础信息** | 小区名称、地址、总户数、商户数 | 了解小区规模 |
| **配电信息** | 配电房位置、数量、供电方式 | 规划用电检查 |
| **联系人** | 客户经理、社区联系人、物业电话 | 现场联系 |
| **历史记录** | 上次驻点时间、驻点次数 | 确定工作重点 |
| **群信息** | 是否建立微信群、群名称 | 确认入群状态 |

**产出**：小区信息摘要

**工具**：WPS多维表-小区台账

#### 步骤2：获取重点客户清单

**执行时间**：驻点前1天上午

**操作内容**：
1. 系统自动推送重点客户清单（企业微信）
2. 查看并确认清单内容
3. 标记重点关注客户

**重点客户类型**：

| 类型 | 识别标准 | 服务策略 |
|------|---------|---------|
| **敏感客户** | 近12月投诉次数≥1次 | 主动回访、了解满意度 |
| **频繁停电客户** | 近12月停电次数≥3次 | 解释说明、优先保障 |
| **特殊群体** | 独居老人、残疾人等 | 上门关怀、安全检查 |
| **重要客户** | 医院、学校、政府等 | 优先服务、重点保障 |

**产出**：重点客户走访计划

**工具**：企业微信推送、WPS多维表-客户台账

#### 步骤3：查询历史驻点记录

**执行时间**：驻点前1天上午

**操作内容**：
1. 查询该小区上次驻点记录
2. 查看上次工作内容
3. 确认遗留问题

**查询内容**：

| 信息类型 | 具体内容 | 用途 |
|---------|---------|------|
| **上次时间** | 上次驻点日期 | 确定频次 |
| **工作内容** | 公示、入群、走访、检查等 | 避免重复 |
| **发现问题** | 隐患、客户诉求 | 跟踪整改 |
| **下次建议** | 上次建议 | 参考执行 |

**产出**：历史遗留问题清单

**工具**：WPS多维表-驻点工作台账

### 3.2 详细步骤

#### 步骤1：公示牌张贴/检查

**执行时间**：到达小区后首先进行

**预计时长**：10分钟

**操作流程**：

```
公示牌工作检查清单

□ 1. 查看公示牌位置
   - 是否位于显眼位置
   - 是否易于客户查看

□ 2. 检查公示牌状态
   - 是否完好无损
   - 信息是否清晰可读
   - 是否有遮挡

□ 3. 检查公示内容
   - 客户经理姓名是否正确
   - 联系电话是否正确
   - 服务承诺是否完整

□ 4. 处理措施
   - 如未张贴：选择合适位置张贴
   - 如已损坏：更换新公示牌
   - 如信息有误：更正信息

□ 5. 拍照记录
   - 拍摄公示牌照片（近景+远景）
   - 记录张贴位置

□ 6. 记录工作
   - 在"驻点工作台账"中记录
   - 勾选"公示牌张贴"
```

**产出**：公示牌照片、工作记录

**工具**：手机相机、驻点工作台账

#### 步骤2：加入/检查微信群

**执行时间**：公示牌后进行

**预计时长**：10分钟

**操作流程**：

```
微信群工作检查清单

□ 1. 确认微信群状态
   - 查询是否已加入小区微信群
   - 如已加入：检查群活跃度

□ 2. 如未加入：申请入群
   - 联系物业或社区网格员
   - 说明入群目的（服务居民）
   - 申请加入业主群

□ 3. 入群后操作
   - 修改群昵称为"武侯供电-客户经理XXX"
   - 发布自我介绍：
     "各位邻居好，我是武侯供电中心客户经理XXX，
      负责咱们小区的用电服务。如有用电问题，
      欢迎随时联系我，电话：138XXXXXXXX"
   - 发送客户经理联系卡图片

□ 4. 群管理
   - 开启消息免打扰（避免干扰）
   - 置顶群聊（便于查看）
   - 备注群信息（小区名称+群类型）

□ 5. 拍照记录
   - 截图群成员列表
   - 截图自我介绍消息

□ 6. 记录工作
   - 在"驻点工作台账"中记录
   - 勾选"加入微信群"
```

**产出**：群聊截图、工作记录

**工具**：企业微信、手机截图

#### 步骤3：重点客户走访

**执行时间**：入群后进行

**预计时长**：30-60分钟（视客户数量而定）

**操作流程**：

```
重点客户走访流程

【走访前准备】
□ 1. 查看重点客户清单
   - 确认走访对象
   - 了解客户背景信息

□ 2. 确定走访顺序
   - 按地理位置排序（避免来回奔波）
   - 优先走访敏感客户和频繁停电客户

□ 3. 准备走访材料
   - 客户经理联系卡
   - 安全用电宣传单
   - 工作记录表

【走访中执行】
对于每个客户：

□ 1. 自我介绍
   "您好，我是武侯供电中心客户经理XXX，
    今天来咱们小区驻点服务，特意来看看您。"

□ 2. 了解客户情况
   - 近期用电是否正常？
   - 有没有遇到什么用电问题？
   - 对我们的服务有什么建议？

□ 3. 针对性服务

   【敏感客户】
   - 询问上次投诉问题是否解决
   - 了解客户满意度
   - 说明改进措施
   - 感谢客户的理解和支持

   【频繁停电客户】
   - 解释停电原因
   - 说明改进措施
   - 承诺优先保障
   - 留下联系电话

   【特殊群体】
   - 检查室内用电安全
   - 讲解安全用电知识
   - 留下联系方式
   - 承诺定期上门服务

   【重要客户】
   - 了解用电需求
   - 介绍保障措施
   - 建立定期沟通机制

□ 4. 宣传告知
   - 发放客户经理联系卡
   - 介绍服务渠道（微信群、电话）
   - 宣传安全用电知识

□ 5. 记录反馈
   - 记录客户诉求
   - 记录客户建议
   - 记录需要跟进的问题

□ 6. 礼貌道别
   "感谢您对我们工作的支持，有问题随时联系我，
    祝您生活愉快！"

【走访后记录】
□ 1. 填写走访记录
   - 客户编号、姓名
   - 走访时间
   - 走访内容摘要
   - 客户反馈
   - 需跟进事项

□ 2. 拍照记录
   - 与客户合影（经客户同意）
   - 拍摄走访现场照片

□ 3. 更新客户信息
   - 在WPS多维表中更新"上次走访时间"
   - 更新"走访次数"
   - 更新"服务记录"
```

**产出**：走访记录、现场照片、客户反馈清单

**工具**：驻点工作台账、手机相机、WPS多维表

#### 步骤4：配电房检查

**执行时间**：客户走访后进行

**预计时长**：30分钟

**操作流程**：

```
配电房检查流程

【检查前准备】
□ 1. 查询配电房信息
   - 查看上次检查记录
   - 了解历史隐患

□ 2. 联系进入
   - 联系物业或社区
   - 说明检查目的
   - 获取配电房钥匙

【检查中执行】

□ 1. 环境检查
   □ 配电房位置是否便于进出
   □ 通道是否畅通
   □ 门窗是否完好
   □ 门锁是否正常
   □ 是否有渗水、积水
   □ 通风是否良好
   □ 照明是否正常
   □ 温度是否异常

□ 2. 设备检查
   □ 变压器
     - 外观是否完好
     - 声音是否正常
     - 温度是否正常
     - 油位是否正常（油变）
     - 标识是否清晰
   
   □ 开关柜
     - 外观是否完好
     - 指示灯是否正常
     - 仪表指示是否正常
     - 开关位置是否正确
     - 标识是否清晰
   
   □ 电缆
     - 外观是否完好
     - 是否有发热
     - 标识是否清晰
   
   □ 接地装置
     - 连接是否牢固
     - 标识是否清晰

□ 3. 安全设施检查
   □ 灭火器
     - 是否在有效期内
     - 压力是否正常
     - 数量是否充足
   
   □ 绝缘垫
     - 是否完好
     - 铺设是否规范
   
   □ 警示标识
     - 是否齐全
     - 是否清晰
   
   □ 防小动物设施
     - 是否完好
     - 是否有效

□ 4. 隐患记录
   - 发现的问题
   - 隐患等级判断
   - 整改建议

【检查后工作】

□ 1. 拍照记录
   - 配电房外观
   - 变压器
   - 开关柜
   - 发现的问题

□ 2. 填写检查记录
   - 检查时间
   - 检查人员
   - 设备状态
   - 发现问题
   - 整改建议

□ 3. 告知物业/社区
   - 通报检查结果
   - 告知发现的隐患
   - 提出整改建议

□ 4. 记录工作
   - 在"驻点工作台账"中记录
   - 勾选"配电房检查"
   - 填写"隐患发现数"
   - 填写"隐患描述"

□ 5. 更新知识库（如采集到新信息）
   - 更新设备信息
   - 更新隐患记录
```

**产出**：配电房检查记录、现场照片、隐患清单

**工具**：配电房检查表、手机相机、驻点工作台账

#### 步骤5：应急资源信息采集

**执行时间**：配电房检查后进行

**预计时长**：20分钟

**操作流程**：

```
应急资源信息采集流程

【信息核查】
□ 1. 查询已有信息
   - 查看知识库中是否已有该小区应急资源信息
   - 如有：核实信息是否准确
   - 如无：开展信息采集

【信息采集】

□ 1. 应急电源信息
   □ 应急发电车接入点位置
     - 具体位置描述
     - 距离配电房的距离
     - 进入路径
   
   □ 接入容量
     - 可用容量（kW）
     - 接入条件
   
   □ 接入操作
     - 操作步骤
     - 注意事项
   
   □ 拍照记录
     - 接入点全景
     - 接入点细节

□ 2. 抢修设备信息
   □ 抢修设备存放位置
   □ 设备清单
   □ 设备状态
   □ 取用方式

□ 3. 应急人员信息
   □ 抢修队伍联系人
   □ 联系电话
   □ 响应时间

□ 4. 应急通道信息
   □ 车辆通道
     - 宽度
     - 限高
     - 进入方式（是否需要开门）
   
   □ 人员通道
     - 快速进入路径
     - 门禁情况
   
   □ 拍照记录
     - 通道照片

□ 5. 物业应急资源
   □ 物业应急联系人
   □ 物业应急电话
   □ 物业应急物资（如发电机）

【信息记录】

□ 1. 填写采集表单
   - 应急电源信息
   - 抢修设备信息
   - 应急人员信息
   - 应急通道信息
   - 物业应急资源

□ 2. 拍照记录
   - 接入点照片
   - 通道照片
   - 其他相关照片

□ 3. 录入知识库
   - 更新应急资源库
```
