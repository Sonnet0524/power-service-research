### 2.2 企业微信使用指南

#### 功能1：接收重点客户清单推送

**操作步骤**：
1. 确保企业微信已登录
2. 驻点前1天查看消息
3. 点击推送消息查看详情
4. 收藏或截图保存

**注意事项**：
- 保持消息通知开启
- 及时查看消息
- 如有疑问及时反馈

#### 功能2：加入小区微信群

**操作步骤**：
1. 联系物业或社区网格员
2. 说明入群目的
3. 扫码或接受邀请入群
4. 修改群昵称
5. 发布自我介绍

**群昵称规范**：
```
武侯供电-客户经理XXX
```

**自我介绍模板**：
```
各位邻居好，我是武侯供电中心客户经理XXX，
负责咱们小区的用电服务。如有用电问题，
欢迎随时联系我：
电话：138XXXXXXXX
微信：同上
工作时间：工作日8:30-17:30
```

#### 功能3：发布通知信息

**操作步骤**：
1. 进入小区微信群
2. 点击输入框
3. 输入通知内容
4. 可配图（如停电通知图）
5. 发送

**通知模板**：

**停电通知**：
```
【停电通知】
尊敬的业主：
因设备检修，计划于XX月XX日XX:XX-XX:XX停电，
停电范围：X栋-X栋
请您提前做好准备，给您带来不便敬请谅解。
如有疑问请联系我：138XXXXXXXX
```

**安全提示**：
```
【安全用电提示】
近期气温升高，用电负荷增大，请大家注意：
1. 避免大功率电器同时使用
2. 定期检查线路和插座
3. 发现异常及时联系
客户经理：XXX 138XXXXXXXX
```

### 2.3 WPS多维表使用指南

#### 功能1：查询客户信息

**操作步骤**：
1. 打开WPS Office APP
2. 进入"多维表"
3. 选择"客户台账"
4. 使用搜索功能（姓名/电话/地址）
5. 查看客户详情

**查询技巧**：
- 支持模糊搜索
- 可筛选客户类型
- 可查看历史服务记录

#### 功能2：录入驻点工作记录

**操作步骤**：
1. 打开WPS Office APP
2. 进入"多维表"
3. 选择"驻点工作台账"
4. 点击"+"新建记录
5. 填写各项信息
6. 保存

**快速录入技巧**：
- 小区名称、客户经理自动填充
- 日期自动填充当前日期
- 可使用选择框快速勾选

#### 功能3：上传现场照片

**操作步骤**：
1. 在表单中找到"现场照片"字段
2. 点击"+"或相机图标
3. 选择"拍照"或"从相册选择"
4. 选择或拍摄照片
5. 确认上传

**照片管理建议**：
- 现场及时拍照
- 照片命名规范（日期+内容）
- 定期整理照片

### 2.4 手机相机使用指南

#### 拍照规范

| 拍摄对象 | 拍摄要求 | 用途 |
|---------|---------|------|
| **公示牌** | 近景+远景，清晰可见 | 工作记录 |
| **微信群** | 截图群成员列表+自我介绍 | 工作记录 |
| **客户走访** | 与客户合影（经同意） | 工作佐证 |
| **配电房** | 全貌+设备细节+问题点 | 检查记录 |
| **隐患** | 全景+细节+参照物 | 整改跟踪 |
| **应急资源** | 接入点+通道+标识 | 知识库建设 |

## 五、补充内容四：风险预案

### 5.1 风险分类与预案

#### 风险1：客户不配合

**场景**：
- 客户拒绝走访
- 客户不配合检查
- 客户态度恶劣

**预案**：
```
1. 保持礼貌和专业
2. 说明来意和服务目的
3. 不强求，记录情况
4. 后续通过其他渠道沟通（电话、微信）
5. 如多次不配合，向供电所汇报
```

#### 风险2：配电房无法进入

**场景**：
- 物业不配合
- 钥匙无法获取
- 配电房被占用

**预案**：
```
1. 提前与物业沟通，获取支持
2. 准备供电所介绍信
3. 如无法进入，记录原因
4. 联系供电所协调
5. 安排下次检查
```

#### 风险3：发现重大隐患

**场景**：
- 发现严重设备缺陷
- 发现重大安全隐患

**预案**：
```
1. 立即拍照记录
2. 告知物业/社区风险
3. 立即向供电所报告
4. 配合制定应急措施
5. 跟踪整改直至消除
```

#### 风险4：客户投诉升级

**场景**：
- 客户对服务不满
- 客户投诉至95598
- 客户向媒体反映

**预案**：
```
1. 立即向供电所报告
2. 主动联系客户安抚
3. 了解诉求和问题
4. 制定解决方案
5. 及时反馈处理结果
6. 总结经验教训
```

#### 风险5：信息泄露风险

**场景**：
- 客户信息泄露
- 工作记录丢失
- 系统被攻击

**预案**：
```
1. 立即报告信息泄露情况
2. 评估影响范围
3. 采取补救措施
4. 加强信息安全管理
5. 追究责任（如人为）
```

### 5.2 应急响应流程

```
发现问题
  ↓
评估风险等级
  ↓
┌─────────────────────────────────────┐
│ 低风险    │ 中风险    │ 高风险     │
│ 自行处理  │ 报告主管  │ 立即上报   │
│ 记录备案  │ 协同处理  │ 启动预案   │
└─────────────────────────────────────┘
  ↓
处理执行
  ↓
结果反馈
  ↓
总结归档
```

## 七、补充内容六：常见问题解答（FAQ）

### Q1：驻点频率如何确定？

**A**：根据小区类型和客户需求确定：
- 新建小区：前3个月每月1次，之后每季度1次
- 老旧小区：每季度至少1次
- 敏感客户集中小区：每月1次
- 其他小区：每半年至少1次

### Q2：如何确定重点客户？

**A**：系统自动识别+人工标记：
- 系统自动识别：近12月停电≥3次、投诉≥1次
- 人工标记：独居老人、残疾人、重要用户等

### Q3：配电房检查发现隐患如何处理？

**A**：
1. 记录隐患详细信息（照片+描述）
2. 评估隐患等级（紧急/重要/一般）
3. 告知物业/社区风险
4. 向供电所报告
5. 跟踪整改进度
6. 整改完成后验证

### Q4：客户拒绝走访怎么办？

**A**：
1. 尊重客户意愿，不强求
2. 记录拒绝原因
3. 尝试电话沟通
4. 后续通过微信群保持联系
5. 如客户有诉求，及时响应

### Q5：信息录入错误如何修改？

**A**：
1. WPS多维表支持修改
2. 找到对应记录
3. 点击编辑修改
4. 保存更新
5. 如已生成统计，需重新计算

### Q6：社网共建如何推进？

**A**：
1. 选择配合度高的社区试点
2. 与社区党组织对接
3. 签订合作协议
4. 设立服务驿站
5. 开展联合服务
6. 总结经验后推广

### Q7：知识库数据谁来维护？

**A**：
- 客户经理：驻点时采集和更新
- 供电所：定期核查和补充
- 系统管理员：数据备份和安全

### Q8：如何提升工作效率？

**A**：
1. 充分利用信息前置机制
2. 提前规划工作路线
3. 熟练使用数字化工具
4. 做好工作记录和总结
5. 定期复盘优化流程
