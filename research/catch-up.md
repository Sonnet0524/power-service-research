#### 1.3 ⭐ 驻点工作真实形态（Session-002重要发现）

```
实际情况：
├── 形式：搭棚子，几个人一起
├── 内容：宣传、交流
├── 时间：一次半天到一天
├── 频率：每周1-2个小区
└── 重复：很久才去第二次

核心特征：流动式服务，不是长期驻扎
```

**核心原则调整**：
- 时间有限 → 筛选"一次可完成"的工作
- 很久才去第二次 → 信息驱动，而非关系驱动
- 必须抓住重点 → 清单指导，精准聚焦

## 📌 重要结论

### 核心原则

1. **时间有限** → 筛选"一次可完成"的工作
2. **很久才去第二次** → 信息驱动，而非关系驱动
3. **必须抓住重点** → 清单指导，精准聚焦
4. **一次见效** → 每次驻点都要有成效

### 工作重点

- 极高优先级：公示、加入群、重点客户走访、配电房检查
- 清单价值：指导客户经理在有限时间内抓住重点
- 数字化核心：快速查询 + 一次录入
